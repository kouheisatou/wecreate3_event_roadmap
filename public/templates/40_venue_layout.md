## 動線計画

### 参加者の動線
```
【入場】
入口 → 受付 → クローク（荷物預け） → メインホール
                  ↓
              トイレ・休憩スペース
                  ↓
【懇親会】
メインホール → ケータリングエリア
                  ↓
【退場】
クローク → 受付（アンケート） → 出口
```

### スタッフの動線
```
スタッフ入口 → バックヤード → 各持ち場
```

### 登壇者の動線
```
スタッフ入口 → 控え室 → ステージ袖 → ステージ → 控え室
```
