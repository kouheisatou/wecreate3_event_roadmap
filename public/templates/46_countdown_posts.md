# カウントダウン投稿テンプレート

## カウントダウン投稿の目的

1. **FOMO（取り残される不安）の喚起**: 「残りわずか」を強調し、申込を促進
2. **リマインド効果**: すでに申し込んだ参加者へのリマインド
3. **盛り上げ**: SNSでの注目度を高め、トレンド入りを狙う
4. **登壇者・スポンサーの紹介**: カウントダウンと合わせて情報発信

#### Twitter/X 投稿例
```

🗓 XX月XX日（X）XX:XX〜
📍 [会場名]
🎤 登壇者: [名前], [名前], [名前]

まだ間に合います！
お申し込みはこちら👇
[URL]

#[イベント名] #カウントダウン #エンジニア
```

#### 画像案
- カウントダウン数字「14」を大きく表示
- イベントビジュアルを背景に
- 申込URLをQRコードで掲載

## カウントダウン画像のデザイン案

### デザインの要素
1. **カウントダウン数字**: 大きく目立つように
2. **イベント情報**: 日時・会場・タイトル
3. **申込URL**: QRコード or 短縮URL
4. **ビジュアル**: イベントのキービジュアル
5. **ブランドカラー**: イベントの統一感

### デザインツール
- **Canva**: テンプレート豊富、初心者向け
- **Figma**: プロ仕様、細かい調整可能
- **Adobe Photoshop/Illustrator**: 本格的なデザイン

### テンプレート例
```
━━━━━━━━━━━━━━━━━━━━━━━━
            あと X 日！
━━━━━━━━━━━━━━━━━━━━━━━━

      [イベント名]

    XX月XX日（X） XX:XX〜
        [会場名]

    [QRコード or URL]

━━━━━━━━━━━━━━━━━━━━━━━━
```

## 投稿効果を高めるコツ

### 1. 投稿時間を最適化
- **平日**: 12:00〜13:00（昼休み）、20:00〜22:00（夜）
- **休日**: 10:00〜11:00、20:00〜21:00

### 2. 画像を必ず添付
- 画像付き投稿はエンゲージメント率が2倍

### 3. ハッシュタグは1〜3個
- 多すぎるとスパムに見える
- `#[イベント名]` は必須

### 4. 登壇者・スポンサーをメンション
- `@username` でメンションすると、拡散してもらえる可能性UP

### 5. エンゲージメントを促す
- 「RTお願いします！」
- 「参加予定の方は♥️してください！」
