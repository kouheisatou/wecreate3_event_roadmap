# 開催レポートテンプレート

## 開催レポートの目的

1. **成果の可視化**: イベントの成果を対外的に発信
2. **コミュニティ形成**: 参加できなかった人への情報共有
3. **次回への布石**: 次回イベントへの期待値向上
4. **SEO対策**: ブログ記事としてWebサイトへの流入増加

---

## 公開媒体

- [ ] 公式ブログ / note
- [ ] Medium
- [ ] Qiita / Zenn
- [ ] 社内ブログ（会社主催の場合）
- [ ] SNS投稿（Twitter/X, LinkedIn）
