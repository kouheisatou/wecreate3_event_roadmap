# 会場候補地比較表

## 会場選定の要件定義
まず、必須条件と希望条件を明確にします。

### 必須条件（Must Have）
- [ ] 開催希望日時に空きがある
- [ ] 収容人数: **最低XX名〜最大XX名**
- [ ] 予算上限: **XX万円以内（税込）**
- [ ] Wi-Fi完備（目安: 100Mbps以上）
- [ ] プロジェクター・スクリーン完備
- [ ] アクセス: **主要駅から徒歩XX分以内**
- [ ] 飲食持ち込み可 / ケータリング手配可

### 希望条件（Nice to Have）
- [ ] マイク・音響設備の質が高い
- [ ] 受付スペースが広い
- [ ] 配信用の照明・機材が充実
- [ ] バックヤード（控え室）がある
- [ ] トイレが近い・数が多い
- [ ] おしゃれな内装（写真映え）
- [ ] 前日設営が可能
