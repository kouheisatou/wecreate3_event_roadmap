# スタッフ最終ミーティング アジェンダ

## ミーティング基本情報

| 項目 | 内容 |
|------|------|
| **ミーティング名** | [イベント名] スタッフ最終MTG |
| **日時** | 20XX年XX月XX日（X） XX:XX〜XX:XX（前日 or 当日開場前） |
| **場所** | [会場名] バックヤード / オンライン |
| **参加者** | 運営スタッフ全員（XX名） |
| **所要時間** | 30〜45分 |

---

## ミーティングの目的

1. **全員の意識統一**: 当日の流れ、目標を全員で共有
2. **最終確認**: 各チームの準備状況、懸念点の確認
3. **士気向上**: チーム全体の士気を高め、一体感を醸成

---

### 4. 緊急時の対応確認（5分）
**担当**: 副統括

#### 緊急連絡フロー
```
トラブル発生
   ↓
各チームリーダーへ報告
   ↓
副統括へエスカレーション
   ↓
統括責任者へ最終判断依頼
```

#### 主なトラブルシナリオ
| トラブル | 第一対応者 | 対応方法 |
|---------|----------|---------|
| 参加者の体調不良 | 誘導チーム | AED確認、必要時119 |
| 音響トラブル | 機材チーム | バックアップマイクへ切り替え |
| 登壇者の遅刻 | 登壇者サポート | タイムテーブル変更検討 |
| 配信トラブル | 配信チーム | バックアップ回線へ切り替え |

#### 緊急連絡網の確認
- [ ] 全員が緊急連絡網を持っている
- [ ] トランシーバーの動作確認済み
- [ ] Slackグループに全員参加済み

### 5. 役割分担の再確認（5分）
**担当**: 統括責任者

#### スタッフ配置の確認
各スタッフが自分の配置場所を再確認する。

**質問**:
- 「自分がどこに配置されるか把握していますか？」
- 「自分の役割を理解していますか？」

#### 休憩ローテーション
- 本編中の休憩時間（10分）に半数ずつ交代で休憩
- 休憩場所: バックヤード

### 6. 注意事項・お願い（5分）
**担当**: 統括責任者

#### 当日のマインドセット
```
✅ 笑顔で対応しましょう
✅ 参加者が主役です
✅ 困ったときはすぐに相談
✅ チーム全体でサポートし合う
✅ 楽しむ余裕を持つ
```

#### 禁止事項
```
❌ スマホでの私用連絡（緊急時を除く）
❌ 参加者の前での愚痴・批判
❌ 勝手な判断での行動（必ずリーダーに相談）
```

#### 忘れ物がないか確認
- [ ] スタッフパス
- [ ] スケジュール表
- [ ] 緊急連絡網
- [ ] 筆記用具
- [ ] 飲料水

---

### 7. 質疑応答（5分）
**担当**: 統括責任者

#### 質問受付
```
何か質問や不安なことはありますか？
些細なことでも構いません。
```

**よくある質問**:
- Q. 遅刻者が来たらどうしますか？
  - A. 受付チームが対応し、誘導チームが静かに誘導します。
- Q. 参加者から質問されたら？
  - A. 分からないことは「確認します」と伝え、リーダーに相談してください。
- Q. トイレ休憩はどうしますか？
  - A. セッションの休憩時間（10分）に交代で休憩してください。

## 備考・特記事項
- ミーティングは簡潔に、30分以内で終わらせる
- 長話は避け、要点のみを伝える
- 質問は積極的に受け付ける
- 士気を高めるため、明るい雰囲気を心がける
