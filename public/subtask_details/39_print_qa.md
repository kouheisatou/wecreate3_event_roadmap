# 印刷物入稿・検品

## タスク概要
パンフレット、バナー、ポスターなどの制作物を印刷会社に入稿し、納品後に検品を行います。

## 入稿フロー

### Step 1: データ最終確認
- [ ] 誤字脱字チェック
- [ ] ロゴ・画像の確認
- [ ] 日時・会場情報の確認
- [ ] チーム内レビュー完了
- [ ] スポンサー企業にロゴ確認（該当する場合）

### Step 2: 入稿データ準備
- [ ] カラーモードをCMYKに変換
- [ ] フォントをアウトライン化
- [ ] 画像を埋め込み
- [ ] 塗り足しを設定（各辺3mm）
- [ ] 解像度を300dpiに設定
- [ ] ファイル形式を.ai or PDFに変換

### Step 3: 印刷会社に事前確認
- [ ] 入稿データを送付
- [ ] データに問題がないか確認依頼
- [ ] 修正点があれば修正
- [ ] 最終確認OK

### Step 4: 正式入稿
- [ ] 印刷会社の入稿フォームから入稿
- [ ] 入稿完了メールを受信
- [ ] 納期を確認
- [ ] 決済完了

### Step 5: 校正確認（必要な場合）
- [ ] 印刷会社から校正PDFを受領
- [ ] 内容を確認
- [ ] 問題なければ承認
- [ ] 修正があれば再校正依頼

- 全体と拡大写真を撮影
- 複数枚撮影

## Tips
- **入稿前に印刷会社に確認**: データに問題がないか必ず事前確認
- **余裕を持った納期**: イベント3日前までに納品されるように発注
- **検品は複数人で**: 見落としを防ぐため、複数人で検品
- **写真を記録**: 検品時の写真を撮影し、記録として保管

## 所要時間目安
4時間（入稿準備1h + 検品2h + 対応1h）
