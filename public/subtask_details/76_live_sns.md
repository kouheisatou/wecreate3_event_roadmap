# リアルタイムSNS発信

## タスク概要
イベントの様子をリアルタイムでSNSに投稿し、盛り上がりを可視化します。参加できなかった人にもイベントの熱気を届け、次回の参加意欲を高めます。

## 目的
- イベントの盛り上がりをリアルタイムで伝える
- 参加できなかった人にも雰囲気を届ける
- ハッシュタグをトレンド入りさせる

## 実施手順
1. **投稿計画の策定**（1時間）
   - 投稿すべきタイミング（開場、各セッション開始、懇親会等）をリスト化
   - 投稿担当者を配置
   - ハッシュタグを統一

2. **テンプレート作成**（30分）
   - 各シーンの投稿テンプレートを用意
   - 画像サイズやキャプションの方針を決定

3. **当日投稿**（イベント中）
   - 開場
   - 各セッション開始
   - 名言・印象的な発言
   - 懇親会の様子
   - 終了

4. **参加者投稿の拡散**（イベント中・後）
   - 参加者のハッシュタグ投稿をリツイート・シェア
   - 参加者との交流

## 成果物
- SNS投稿（X, Instagram等）10-20投稿
- ハッシュタグ投稿数の記録

## 投稿ガイドライン

### 写真
- [ ] 解像度は適切か（スマホ写真でOK）
- [ ] 参加者の顔が識別できる場合、許可を確認済みか
- [ ] スポンサーロゴが適切に写り込んでいるか

### キャプション
- [ ] ハッシュタグは必ず付ける（#イベント名 #WeCreate3）
- [ ] 登壇者のアカウントをメンション（@username）
- [ ] 絵文字で親しみやすく🎉
- [ ] URLは短縮URLを使用

### タイミング
- [ ] セッション中の投稿は控えめに（集中の妨げにならないよう）
- [ ] 休憩時間や懇親会は積極的に投稿
- [ ] リアルタイム性を重視（終了後ではなく、その場で）

## Tips
- **リアルタイム性が命**: イベント終了後ではなく、その場で投稿
- **参加者を巻き込む**: 「#イベント名 で投稿してください！」と呼びかけ
- **登壇者をメンション**: 登壇者のアカウントをメンションすると拡散されやすい
- **絵文字で親しみやすく**: 固い文章よりも絵文字を使って親しみやすく
- **写真は複数枚**: 1投稿に2-4枚の写真を載せると、情報量が増える
- **ストーリーズも活用**: InstagramやX（旧Twitter）のストーリーズも活用
