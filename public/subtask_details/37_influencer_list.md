# インフルエンサー候補リストを作成する

## タスク概要

ターゲット層に影響力のあるインフルエンサーをリストアップします。フォロワー数だけでなく、エンゲージメント率、専門分野、ターゲット層との親和性を考慮して選定します。マイクロインフルエンサー（フォロワー1万〜10万人）は、エンゲージメント率が高く、費用対効果が良い傾向があります。

## 目的

- ターゲット層に影響力のあるインフルエンサーを特定する
- フォロワー数、エンゲージメント率、専門分野を考慮する
- マイクロインフルエンサーも含め、幅広くリストアップする
- 営業メール送付の準備をする

## 実施手順

1. **インフルエンサーの選定基準を決める**（15分）
   - フォロワー数（1万〜10万人のマイクロインフルエンサーが効果的）
   - エンゲージメント率（3%以上が目安）
   - 専門分野（AI、キャリア、ビジネス、エンジニアリングなど）
   - ターゲット層との親和性（20代のフォロワーが多いか）
   - 過去の投稿内容（イベント紹介の実績があるか）

2. **プラットフォームを決める**（5分）
   - X（Twitter）: ビジネス系インフルエンサーが多い
   - Instagram: ビジュアル重視、ライフスタイル系インフルエンサーが多い
   - LinkedIn: ビジネス系、キャリア系インフルエンサーが多い
   - YouTube: 動画コンテンツ、解説系インフルエンサーが多い

3. **インフルエンサーの検索**（60分）
   - ハッシュタグ検索（#AI #キャリア #20代 #エンジニアなど）
   - X（Twitter）のリスト機能を活用
   - Instagramの「おすすめ」機能を活用
   - Google検索（「AI インフルエンサー」「キャリア インフルエンサー」など）
   - インフルエンサーマーケティングツール（後述）の利用

4. **候補者をリスト化**（30分）
   - 候補者の名前、SNSアカウント、フォロワー数、エンゲージメント率、専門分野を記録
   - Google SpreadsheetやNotionにリスト化
   - 10〜20名程度リストアップ

5. **優先順位をつける**（15分）
   - エンゲージメント率が高い順に並べ替え
   - ターゲット層との親和性が高い順に並べ替え
   - 上位5〜10名を優先的に営業する

## 成果物

- インフルエンサー候補リスト（名前、SNSアカウント、フォロワー数、エンゲージメント率、専門分野、優先順位）

### エンゲージメント率の計算方法

```markdown
# エンゲージメント率の計算方法

エンゲージメント率 = （いいね数 + コメント数 + リポスト数）÷ フォロワー数 × 100

## 例

- いいね数: 200
- コメント数: 20
- リポスト数: 30
- フォロワー数: 50,000

エンゲージメント率 = （200 + 20 + 30）÷ 50,000 × 100 = 0.5%

## 目安

- **5%以上**: 非常に高い（マイクロインフルエンサーに多い）
- **3〜5%**: 高い
- **1〜3%**: 普通
- **1%未満**: 低い（フォロワー数が多い場合は普通）
```

## Tips

- **マイクロインフルエンサーが効果的**: フォロワー1万〜10万人のマイクロインフルエンサーは、エンゲージメント率が高く、費用対効果が良い
- **エンゲージメント率を重視**: フォロワー数だけでなく、エンゲージメント率（いいね、コメント、リポストの割合）を確認
- **専門分野とターゲット層の一致**: イベントのテーマと、インフルエンサーの専門分野、フォロワー層が一致しているか確認
- **過去の投稿内容を確認**: イベント紹介の実績があるインフルエンサーは、紹介してくれる可能性が高い
- **複数のプラットフォームを活用**: X、Instagram、LinkedIn、YouTubeなど、複数のプラットフォームでリーチを拡大
- **ハッシュタグ検索が有効**: #AI #キャリア #20代 #エンジニア などのハッシュタグで検索すると、関連するインフルエンサーが見つかる
- **インフルエンサーマーケティングツールを活用**: 
  - **X（Twitter）**: TweetDeck、SocialDog
  - **Instagram**: Later、Hootsuite
  - **LinkedIn**: LinkedIn Sales Navigator
- **相互フォローしている人を確認**: イベントに関連する企業やコミュニティと相互フォローしている人は、興味を持ってくれる可能性が高い
