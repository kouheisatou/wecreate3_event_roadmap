# 誘導用のチラシを作成・印刷

## タスク概要

会場への誘導や案内表示用のチラシを作成し、印刷します。駅からのルート、会場入口などに設置することで、参加者が会場を見つけやすくなります。特に、初めて会場に来る参加者にとって、誘導チラシは重要です。

## 目的

- 会場への誘導チラシを作成する
- 駅からのルート、会場入口に設置する
- 参加者が会場を見つけやすくする
- 迷子を防ぐ

## 実施手順

1. **誘導ルートの確認**（15分）
   - 最寄り駅から会場までのルート確認
   - 分かりにくいポイントを特定
   - 設置場所を決定（駅改札口、会場入口など）

2. **チラシのデザイン**（60分）
   - イベントロゴ
   - 「[イベント名] →」という矢印
   - 「会場はこちらです」というメッセージ
   - 地図（駅から会場までのルート）
   - ハッシュタグ

3. **印刷**（30分）
   - A4サイズ、カラー印刷
   - 枚数: 10〜20枚
   - ラミネート加工（雨天対策）

4. **設置**（30分、当日朝）
   - 駅改札口に設置（養生テープで貼り付け）
   - 会場入口に設置
   - 分かりにくいポイントに設置

5. **撤収**（15分、当日終了後）
   - 設置したチラシを全て回収
   - 養生テープの跡を拭き取る

## 成果物

- 誘導チラシ

## チェックリスト

- [ ] 誘導ルートを確認している（最寄り駅から会場まで）
- [ ] 分かりにくいポイントを特定している
- [ ] 設置場所を決定している（駅改札口、会場入口など）
- [ ] チラシをデザインしている（イベントロゴ、矢印、地図、ハッシュタグ）
- [ ] 印刷している（A4、カラー、10〜20枚）
- [ ] ラミネート加工をしている（雨天対策）
- [ ] 当日朝、設置している（駅改札口、会場入口、分かりにくいポイント）
- [ ] 養生テープで貼り付けている（剥がしやすい）
- [ ] 当日終了後、撤収している
- [ ] 養生テープの跡を拭き取っている

## Tips

- **A4サイズが最適**: A4サイズが視認性と持ち運びのバランスが良い
- **矢印を大きく**: 「→」という矢印を大きく表示し、一目で分かるようにする
- **地図を含める**: 駅から会場までのルート地図を含めると、参加者が迷わない
- **ラミネート加工**: 雨天時に備えて、ラミネート加工をすると安心
- **養生テープで貼り付け**: ガムテープではなく、養生テープを使うと、剥がしやすく跡が残らない
- **駅改札口に設置**: 駅改札口に設置することで、最寄り駅に到着した参加者がすぐに見つけられる
- **分かりにくいポイントに設置**: 曲がり角や分岐点など、分かりにくいポイントに設置すると効果的
- **当日朝に設置**: 前日に設置すると、他のイベントのチラシと混同されるため、当日朝に設置
- **撤収を忘れずに**: イベント終了後、必ず撤収し、会場や駅に迷惑をかけないようにする

## 所要時間目安

2.5時間（ルート確認15分 + デザイン60分 + 印刷30分 + 設置30分（当日朝） + 撤収15分（当日終了後））
