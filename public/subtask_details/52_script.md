# 進行台本を作成する

## タスク概要

司会者が使用する進行台本を作成します。オープニング、セッション間の繋ぎ、クロージングなど、具体的なセリフやタイミングを記載します。台本があることで、司会者が安心して進行でき、イベントがスムーズに進みます。

## 目的

- 司会者が使用する進行台本を作成する
- オープニング、セッション間の繋ぎ、クロージングのセリフを明記する
- タイミングや注意事項を記載する
- 司会者が安心して進行できるようにする
- イベントをスムーズに進行させる

## 実施手順

1. **イベントの流れを確認**（10分）
   - タイムテーブルを参照
   - オープニング、各セッション、ランチ、ネットワーキング、クロージングの流れを確認

2. **オープニングのセリフ作成**（30分）
   - 挨拶
   - イベントの目的説明
   - スポンサー・パートナーへの感謝
   - タイムテーブルの説明
   - ハッシュタグの案内

3. **セッション間の繋ぎのセリフ作成**（30分）
   - 登壇者の紹介
   - セッション内容の説明
   - 時間の案内（「このセッションは〇分間です」）
   - Q&Aの案内

4. **クロージングのセリフ作成**（30分）
   - イベントの振り返り
   - 登壇者、スポンサー、参加者への感謝
   - アンケート依頼
   - 次回イベントの告知
   - 締めの挨拶

5. **タイミングと注意事項の追加**（15分）
   - 各セリフのタイミング（「10:30に開始」など）
   - 注意事項（「マイクの音量確認」「プロジェクターの切り替え」など）

6. **台本のレビュー**（15分）
   - 司会者に共有し、フィードバックを収集
   - 読みやすさ、分かりやすさを確認
   - 調整が必要な場合、修正

7. **台本の印刷**（5分）
   - 紙の台本を印刷
   - 司会者に事前配布

## 成果物

- 進行台本（PDF、紙）

## コピー可能なテンプレート

### 進行台本

```markdown
# [イベント名] 進行台本

**開催日**: YYYY/MM/DD（曜日）  
**司会者**: [氏名]  
**作成日**: YYYY/MM/DD

---

## 事前確認

**集合時間**: 9:00  
**リハーサル**: 9:30-10:00

### 持ち物

- この台本
- 筆記用具
- スマートフォン（タイマー用）

### 注意事項

- マイクの音量を事前に確認
- プロジェクターの切り替えタイミングを確認
- タイムテーブルを常に意識
- 笑顔で、参加者を迎える
```

ぜひお立ち寄りください。


（スライドにハッシュタグを表示）


## Tips

- **具体的なセリフを記載**: 「〇〇を説明する」ではなく、実際のセリフを記載することで、司会者が安心して進行できる
- **タイミングを明記**: 「10:30に開始」など、具体的なタイミングを明記
- **注意事項を追加**: 「マイクの音量確認」「プロジェクターの切り替え」など、注意事項を追加
- **笑顔を忘れずに**: セリフの合間に「（笑顔で）」「（会場を見渡し）」などの動作を記載
- **拍手を促す**: 登壇者の紹介後、セッション終了後などに「（拍手を促す）」と記載
- **柔軟に対応**: 台本はあくまでも目安。当日の状況に応じて、柔軟に対応
- **事前リハーサル**: 司会者と一緒にリハーサルを実施し、台本を読み合わせ
- **紙の台本を用意**: スマホで見るよりも、紙の台本の方が見やすい

## 所要時間目安

2.5時間（流れ確認10分 + オープニング作成30分 + 繋ぎ作成30分 + クロージング作成30分 + タイミング追加15分 + レビュー15分 + 印刷5分）
