# タイムテーブルを作成する

## タスク概要

各セッションの開始・終了時間を記載したタイムテーブルを作成します。遅延に耐えられるバッファ時間を設けることで、当日のトラブルに対応できるようにします。タイムテーブルは、スタッフ、登壇者、参加者全員が共有し、イベントを円滑に進行するための重要なツールです。

## 目的

- 各セッションの開始・終了時間を明確にする
- 遅延に耐えられるバッファ時間を設ける
- スタッフ、登壇者、参加者全員が共有できるタイムテーブルを作成する
- イベントを円滑に進行するための指針とする

## 実施手順

1. **イベント全体の構成を決める**（30分）
   - オープニング
   - セッション1（基調講演）
   - セッション2（パネルディスカッション）
   - ランチ休憩
   - セッション3（Q&Aセッション）
   - ネットワーキング
   - クロージング

2. **各セッションの所要時間を決める**（30分）
   - オープニング: 10分
   - セッション1: 30分
   - セッション2: 60分
   - ランチ休憩: 60分
   - セッション3: 30分
   - ネットワーキング: 60分
   - クロージング: 10分
   - 合計: 4時間

3. **バッファ時間を設ける**（15分）
   - 各セッション間に5〜10分のバッファ時間を設ける
   - 登壇者の入れ替え、機材セットアップに時間がかかる場合を想定
   - 遅延が発生した場合、バッファ時間で調整

4. **タイムテーブルの作成**（30分）
   - 開始時間と終了時間を明記
   - セッション名、登壇者名を記載
   - バッファ時間も明記（スタッフ用）

5. **タイムテーブルのレビュー**（15分）
   - 運営メンバーに共有し、フィードバックを収集
   - 登壇者に共有し、時間配分が適切か確認
   - 調整が必要な場合、修正

6. **タイムテーブルの共有**（10分）
   - スタッフにLINEグループやメールで共有
   - 登壇者にメールで共有
   - 参加者にLumaページやSNSで共有

7. **当日の配布**（5分）
   - 紙のタイムテーブルを印刷し、受付で配布
   - スライドでタイムテーブルを表示

## 成果物

- タイムテーブル（PDF、スライド）

- **バッファ時間**: 各セッション間に5〜10分のバッファ時間を設けています。遅延が発生した場合、この時間で調整してください。

- **ネットワーキング**: 参加者が盛り上がっている場合、10分程度延長しても構いません。

### バッファ時間の設定例

```markdown
# バッファ時間の設定

---

## バッファ時間とは？

各セッション間に設ける余裕時間のこと。遅延が発生した場合、この時間で調整します。

---

## 設定例

| セッション間 | バッファ時間 | 理由 |
|------------|------------|------|
| オープニング → セッション1 | なし | 開始時間は厳守 |
| セッション1 → セッション2 | 5分 | 登壇者入れ替え |
| セッション2 → ランチ休憩 | 5分 | 質問が延びた場合に備える |
| ランチ休憩 → セッション3 | なし | 休憩時間で調整可能 |
| セッション3 → ネットワーキング | 5分 | Q&Aが延びた場合に備える |
| ネットワーキング → クロージング | なし | ネットワーキングは多少延長してもOK |

---

## 運用ルール

- 遅延が発生した場合、バッファ時間を使って調整
- バッファ時間を使い切った場合、次のセッションを短縮
- クロージングは最低5分は確保（次回イベント告知のため）

---

**作成日**: YYYY/MM/DD
```

## Tips

- **バッファ時間は必須**: 各セッション間に5〜10分のバッファ時間を設けることで、遅延に対応できる
- **開始時間は厳守**: 受付やオープニングの開始時間は厳守。遅延すると、全体のスケジュールが崩れる
- **Q&Aは柔軟に**: Q&Aセッションは、質問が少ない場合は早めに終了し、多い場合は延長してもOK
- **ネットワーキングは延長可**: 参加者が盛り上がっている場合、ネットワーキングは10分程度延長しても構わない
- **クロージングは最低5分**: 次回イベント告知、アンケート依頼のため、クロージングは最低5分は確保
- **紙のタイムテーブルも配布**: スマホで見られない参加者のために、紙のタイムテーブルも配布
- **スライドで表示**: 会場のスクリーンに、タイムテーブルを常に表示しておくと、参加者が時間を把握しやすい
- **時間厳守の登壇者には事前共有**: 時間に厳しい登壇者には、事前にタイムテーブルを共有し、時間配分を確認

## 所要時間目安

2時間（構成決定30分 + 所要時間決定30分 + バッファ時間設定15分 + タイムテーブル作成30分 + レビュー15分 + 共有10分 + 配布5分）
