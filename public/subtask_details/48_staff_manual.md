# 当日マニュアルを作成する

## タスク概要

受付、誘導、撮影、ブース対応など、各役割の具体的な作業内容をまとめたマニュアルを作成します。スタッフが初めて参加する場合でも、マニュアルを読めば役割を理解し、適切に対応できるようにします。当日の混乱を防ぎ、円滑なイベント運営を実現します。

## 目的

- 各役割の具体的な作業内容を明確にする
- スタッフが初めてでも、適切に対応できるようにする
- 当日の混乱を防ぐ
- トラブル時の対応方法を事前に共有する
- スタッフの不安を軽減する

## 実施手順

1. **役割ごとの作業内容を整理**（30分）
   - 受付スタッフ: 参加者のチェックイン、名札配布
   - 誘導スタッフ: 会場への誘導、座席案内
   - 撮影スタッフ: イベントの様子を撮影（写真・動画）
   - ブース対応スタッフ: スポンサーブースの対応
   - 音響・映像スタッフ: マイク、プロジェクター、Zoom配信の操作
   - ケータリング対応スタッフ: 軽食・ドリンクの配置、補充

2. **マニュアルの構成を決める**（15分）
   - 全体の流れ（集合時間、リハーサル、本番、撤収）
   - 役割別の詳細マニュアル
   - よくあるトラブルと対応方法
   - 緊急連絡先

3. **マニュアルの作成**（120分）
   - 各役割ごとに、具体的な作業内容を記載
   - チェックリスト形式で、やるべきことを明記
   - よくある質問（FAQ）を記載
   - 図やイラストを追加（受付の配置図、会場のレイアウトなど）

4. **マニュアルのレビュー**（30分）
   - 運営メンバーに共有し、フィードバックを収集
   - 分かりにくい箇所がないか確認
   - 誤字脱字がないか確認

5. **マニュアルの配布**（10分）
   - Google DriveまたはNotionにアップロード
   - LINEグループやメールで、全スタッフに共有
   - 前日までに読んでおくよう依頼

6. **当日の配布**（5分）
   - 紙のマニュアルも用意し、当日配布
   - スマホで見られない場合に備える

## 成果物

- 当日マニュアル（PDF、紙）

## 役割別マニュアル

### 1. 受付スタッフ

#### 作業内容

1. **受付テーブルの準備**（9:30-10:00）
   - 受付テーブルに名札、参加者リストを配置
   - ペン、スタンプ、配布資料を準備

2. **参加者のチェックイン**（10:00-10:30）
   - 参加者がLumaアプリでQRコードを提示
   - QRコードをスキャン、またはリストで名前を確認
   - 名札を配布
   - 「ようこそ！会場は2階です」と案内

3. **遅刻者対応**（10:30-12:00）
   - 遅刻者が来た場合、同様にチェックイン
   - 「セッションは始まっていますが、お静かにご入場ください」と案内

4. **受付終了**（12:00-）
   - 受付テーブルを片付け
   - 次の役割（誘導やブース対応）へローテーション

#### チェックリスト

- [ ] 受付テーブルに名札、参加者リスト、ペン、スタンプ、配布資料を配置
- [ ] 参加者のQRコードをスキャン、または名前を確認
- [ ] 名札を配布
- [ ] 会場を案内
- [ ] 遅刻者にも同様に対応
- [ ] 受付終了後、テーブルを片付け

#### よくある質問（FAQ）

**Q: 参加者リストに名前がない場合は？**  
A: 統括責任者に確認してください。Lumaで追加登録されている可能性があります。

**Q: 名札を忘れた参加者がいる場合は？**  
A: 予備の名札を手書きで作成してください。

**Q: 遅刻者が多い場合は？**  
A: 誘導スタッフに協力を依頼し、受付と誘導を同時に対応してください。

---

### 2. 誘導スタッフ

#### 作業内容

1. **会場入口での案内**（10:00-10:30）
   - 会場入口に立ち、参加者を会場内に誘導
   - 「会場はこちらです。お荷物は座席の下に置いてください」と案内

2. **座席案内**（10:00-10:30）
   - 参加者が座席に迷っている場合、案内
   - 「前方の席が空いていますよ」など、適宜案内

3. **トイレ・休憩スペースの案内**（随時）
   - 参加者から「トイレはどこですか？」と聞かれたら案内
   - 会場マップを見せて説明

4. **ネットワーキングエリアの案内**（16:00-17:00）
   - ネットワーキングタイムに、参加者をネットワーキングエリアに誘導
   - 「こちらで軽食をお楽しみください」と案内

#### チェックリスト

- [ ] 会場入口に立ち、参加者を誘導
- [ ] 座席案内
- [ ] トイレ・休憩スペースの案内
- [ ] ネットワーキングエリアの案内

#### よくある質問（FAQ）

**Q: 会場が満席の場合は？**  
A: 追加の椅子を会場スタッフに依頼してください。

**Q: 参加者が荷物を置く場所に困っている場合は？**  
A: 「座席の下に置いてください」と案内してください。クロークはありません。

---

### 3. 撮影スタッフ

#### 作業内容

1. **撮影機材の準備**（9:30-10:00）
   - カメラ、スマートフォンの充電確認
   - 予備バッテリー、SDカードの準備

2. **オープニングの撮影**（10:30-11:00）
   - 会場全体の様子を撮影
   - 登壇者の入場を撮影

3. **セッション中の撮影**（11:00-17:00）
   - 登壇者のスライド、表情を撮影
   - 参加者の様子を撮影（後ろから、横から）
   - パネルディスカッションの様子を撮影

4. **ネットワーキングの撮影**（16:00-17:00）
   - 参加者同士の交流の様子を撮影
   - スポンサーブースの様子を撮影

5. **写真のアップロード**（17:30-18:00）
   - Google Driveに写真をアップロード
   - フォルダ名: 「YYYY-MM-DD_[イベント名]_写真」

#### チェックリスト

- [ ] カメラ、スマートフォンの充電確認
- [ ] オープニングの撮影
- [ ] セッション中の撮影（登壇者、参加者）
- [ ] ネットワーキングの撮影
- [ ] 写真をGoogle Driveにアップロード

#### 注意事項

- 参加者の顔が写る場合、後でSNS投稿の許可を取ります（撮影時点では許可不要）
- フラッシュは使用しないでください（登壇者の妨げになります）
- 動画を撮影する場合、音声も録音してください

4. **トラブル対応**（随時）
   - マイクが使えない場合、予備マイクを使用

## よくあるトラブルと対応方法

### 1. 参加者が体調不良

- 対応: 統括責任者に連絡。必要に応じて救急車を呼ぶ

### 2. マイクが使えない

- 対応: 予備マイクを使用。会場スタッフに連絡

### 3. Zoom配信が途切れる

- 対応: Zoomを再起動。Wi-Fiの接続を確認

### 4. 登壇者が遅刻

- 対応: 統括責任者が登壇者に電話。セッションの順序を入れ替え

### 5. 参加者同士のトラブル

- 対応: 統括責任者に連絡。必要に応じて警備員を呼ぶ

## Tips

- **チェックリスト形式**: やるべきことをチェックリスト形式で記載することで、スタッフが漏れなく作業できる
- **FAQ（よくある質問）**: 事前によくある質問を記載しておくことで、当日の質問が減る
- **図やイラスト**: 受付の配置図、会場のレイアウトなどを図示すると、視覚的に分かりやすい
- **トラブル対応を明記**: トラブル時の対応方法を事前に記載しておくと、当日慌てずに済む
- **紙のマニュアルも用意**: スマホで見られない場合に備えて、紙のマニュアルも用意
- **前日までに読んでもらう**: 当日ではなく、前日までに読んでもらうことで、スムーズに作業できる
- **簡潔に**: 長すぎるマニュアルは読まれないため、簡潔に要点をまとめる
- **ポジティブな言葉**: 「〜しないでください」ではなく、「〜してください」とポジティブな言葉で記載
