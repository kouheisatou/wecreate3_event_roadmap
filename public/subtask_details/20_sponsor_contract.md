# スポンサー契約締結・請求

## タスク概要
スポンサー契約の意向が確認できたら、契約書を締結し、請求書を発行し、入金確認を行います。

## コピー可能なテンプレート

```markdown
# スポンサー契約フロー

**イベント名**: Web3 & AI 超会議 2026 Spring

---

## 契約フロー

### Step 1: 契約意向確認
- [ ] 商談で契約の意向を確認
- [ ] プラン・金額を確定
- [ ] 特典内容を最終確認

### Step 2: 契約書作成
- [ ] 契約書テンプレートを修正
- [ ] 企業名、金額、特典内容を記載
- [ ] 社内レビュー（統括、法務担当）

### Step 3: 契約書送付
- [ ] PDFで契約書を送付
- [ ] 押印・返送を依頼
- [ ] 返送期限を設定（2週間）

### Step 4: 契約書締結
- [ ] 先方から押印済み契約書を受領
- [ ] WeCreate3側も押印
- [ ] 双方が原本を保管

### Step 5: 請求書発行
- [ ] 請求書を作成
- [ ] PDFで送付
- [ ] 支払期限を設定（1ヶ月以内）

### Step 6: 入金確認
- [ ] 銀行口座を定期的に確認
- [ ] 入金を確認
- [ ] 領収書を発行
- [ ] お礼メールを送信

### Step 7: 特典提供開始
- [ ] ロゴデータを受領
- [ ] イベントページにロゴ掲載
- [ ] プレスリリースに社名記載
- [ ] 進捗を定期的に報告

---

## 契約書テンプレート

```
協賛契約書

WeCreate3（以下「甲」という）と[企業名]（以下「乙」という）は、
甲が主催する「Web3 & AI 超会議 2026 Spring」（以下「本イベント」という）への
乙の協賛に関し、以下の通り契約を締結する。

第1条（協賛金額）
乙は甲に対し、本イベントへの協賛金として、金[XXX,XXX]円（税込）を支払う。

第2条（支払方法）
1. 乙は、本契約締結後、甲の指定する銀行口座に協賛金を振り込む。
2. 振込手数料は乙の負担とする。
3. 支払期限は、YYYY年MM月DD日とする。

第3条（協賛特典）
甲は乙に対し、以下の特典を提供する。
1. ロゴ掲載（イベントページ、プレスリリース、会場バナー、パンフレット）
2. 登壇枠（XX分）
3. ブース出展
4. SNS投稿（X回）
5. 参加者名簿提供
6. 懇親会参加枠（X名）
7. チケット無料枠（X名）

第4条（ロゴデータ提供）
1. 乙は甲に対し、ロゴデータ（PNG、AI形式）を提供する。
2. 提供期限は、YYYY年MM月DD日とする。

第5条（禁止事項）
乙は、以下の行為を行ってはならない。
1. 本イベントの趣旨に反する行為
2. 他の参加者に迷惑をかける行為
3. 甲の許可なく録音・録画する行為

第6条（キャンセル）
1. 乙が本契約をキャンセルする場合、以下のキャンセル料が発生する。
   - X週間前まで: 無料
   - X週間前〜X日前: 協賛金の50%
   - X日前以降: 協賛金の100%
2. 甲が本イベントを中止する場合、協賛金は全額返金する。

第7条（契約期間）
本契約の有効期間は、本契約締結日から本イベント終了日までとする。

第8条（協議事項）
本契約に定めのない事項については、甲乙協議の上、決定する。

以上、本契約の成立を証するため、本書2通を作成し、甲乙各々記名押印の上、各1通を保有する。

YYYY年MM月DD日

甲: WeCreate3
  代表 [氏名]              印

乙: [企業名]
  [役職] [氏名]           印
```

## 入金確認フロー

### 定期確認
- [ ] 毎週月曜日に銀行口座を確認
- [ ] 入金があれば、企業名と金額を照合
- [ ] スプレッドシートに記録

### 入金確認後
- [ ] お礼メール送信
- [ ] 領収書発行・送付
- [ ] 契約書ファイルに「入金済」と記録
- [ ] 特典提供を開始

### 期限経過後
- [ ] 支払期限を3日過ぎたらリマインドメール送信
- [ ] 1週間過ぎたら電話で確認
- [ ] 2週間過ぎたら統括に報告、対応協議
```

## Tips
- **契約書は弁護士に確認**: 法的リスクを避けるため、契約書は弁護士に確認してもらう
- **入金確認を怠らない**: 定期的に銀行口座を確認
- **リマインドは丁寧に**: 支払い遅延のリマインドは丁寧な言葉遣いで
- **特典提供を忘れずに**: 契約したら特典提供を開始

## 所要時間目安
2時間（契約書作成 + 請求書発行 + 入金確認）× 社数
