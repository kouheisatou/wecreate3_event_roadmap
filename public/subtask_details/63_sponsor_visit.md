# スポンサーお礼訪問

## タスク概要
イベント終了後、スポンサー企業を訪問し、実施報告書を持って感謝を伝え、次回イベントの協賛も提案します。

## コピー可能なテンプレート

```markdown
# スポンサーお礼訪問ガイド

**イベント名**: Web3 & AI 超会議 2026 Spring

---

## お礼訪問の目的

1. **感謝を伝える**: 協賛への感謝を直接伝える
2. **関係構築**: 継続的な関係を築く
3. **フィードバック収集**: イベントへの評価や改善点を聞く
4. **次回提案**: 次回イベントの協賛を提案

---

## 訪問対象企業

### Platinum / Gold スポンサー: 必須訪問
- 高額協賛企業には必ず訪問

### Silver スポンサー: できれば訪問
- 可能であれば訪問、難しければオンラインMTG

### Bronze スポンサー: メール or オンライン
- メールでのお礼 or オンラインMTG

---

## 訪問スケジュール

### イベント終了後1週間以内
- [ ] 実施報告書を作成
- [ ] 訪問候補日を3つ用意
- [ ] 訪問依頼メールを送信

### イベント終了後2-4週間
- [ ] 訪問実施
- [ ] お礼 + 実施報告 + 次回提案

---

## 訪問の流れ（30分）

### 1. 挨拶・感謝（3分）
```
本日はお時間をいただき、ありがとうございます。
WeCreate3の[氏名]です。

改めまして、この度は「Web3 & AI 超会議 2026 Spring」に
Goldスポンサーとしてご協賛いただき、誠にありがとうございました。

おかげさまで、大盛況のうちにイベントを終えることができました。
```

### 2. 実施報告（10分）
```
本日は、イベントの実施報告をお持ちいたしました。

（実施報告書を見せながら）

【参加者数】
目標200名のところ、250名以上の参加者を集めることができました。

【満足度】
アンケートでは、満足度4.3/5.0という高い評価をいただきました。

【御社の露出実績】
プレスリリース配信で累計5,000PV、
SNS拡散で推定リーチ20,000人、
ブース訪問者は約80名でした。

【参加者の声】
「御社の登壇が非常に参考になった」
「御社のブースで話を聞いて、興味を持った」
といった声を多数いただきました。

（写真を見せながら）
こちらがイベント当日の写真です。
御社のブースも多くの参加者で賑わっていました。
```

### 3. フィードバック収集（10分）
```
御社から見て、今回のイベントはいかがでしたでしょうか？

良かった点、改善すべき点など、率直なご意見をいただけますと幸いです。

（フィードバックを聞く）

貴重なご意見をありがとうございます。
次回イベントに活かさせていただきます。
```

### 4. 次回イベントの提案（5分）
```
実は、次回イベント「Web3 & AI 超会議 2026 Fall」を
YYYY年9月に企画しております。

詳細が決まり次第、改めてご連絡させていただきます。

```

### 5. クロージング（2分）
```

引き続きWeCreate3をよろしくお願いいたします。

何かご不明点やご要望がございましたら、

## 想定質問と回答例

### Q1: 今回のイベントで、採用に繋がりましたか?
```
A: アンケートでは、参加者の20%が御社に興味を持ったと回答しています。
   また、ブース訪問者80名のうち、約50名と名刺交換ができましたので、
   今後のフォローアップ次第で採用に繋がる可能性は高いと思います。
```

### Q2: 費用対効果はどうでしたか?
```
A: 推定リーチ20,000人で、協賛金¥500,000ですので、
   CPM（1,000インプレッションあたりのコスト）は¥25となります。
   一般的なWeb広告のCPM（¥500〜1,000）と比較すると、
   非常に費用対効果が高かったと言えます。
```

### Q3: 次回はいつですか?
```
A: 次回は「Web3 & AI 超会議 2026 Fall」を
   YYYY年9月に開催予定です。
   詳細が決まり次第、改めてご連絡させていただきます。
```

### Q4: 次回も同じプランで協賛可能ですか?
```
A: はい、もちろん可能です。
   今回はGoldスポンサー（¥500,000）でしたが、
   次回も同プランでご協賛いただけますと幸いです。

   また、Platinumスポンサー（¥1,000,000）にアップグレードいただくことも可能です。
   ご検討いただけますと幸いです。
```

## Tips
- **訪問は必須**: メールだけでなく、直接訪問することで関係が深まる
- **フィードバックを真摯に聞く**: 改善点を素直に聞く
- **次回提案は控えめに**: 強引な営業にならないよう、控えめに提案
- **議事録を必ず作成**: 訪問後、議事録を作成し、チーム内に共有

## 所要時間目安
2時間（訪問準備30分 + 移動30分 + 訪問30分 + フォロー30分）× 社数
