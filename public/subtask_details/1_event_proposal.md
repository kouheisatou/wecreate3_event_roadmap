# イベント企画書

## タスク概要
イベントの全体像をまとめた企画書を作成します。スポンサー営業や会場交渉、メンバー間の認識統一に使用する重要なドキュメントです。WeCreate3のようなWeb3学生団体では、テクノロジーと社会実装の視点を盛り込むことが重要です。

## 目的
- イベントの目的とコンセプトを明文化する
- ステークホルダー（スポンサー、登壇者、会場）への説明資料とする
- チーム全体の共通認識を形成する

## 実施手順
1. **イベント概要の整理**
   - イベント名、開催日時、場所
   - 対象者、予想参加人数
   - 開催形式（カンファレンス、パネル、ワークショップ等）

2. **背景と目的の言語化**
   - なぜこのイベントが必要なのか
   - 参加者にどんな価値を提供するのか
   - WeCreate3のミッションとの関連性

3. **企画内容の詳細化**
   - セッション構成
   - 登壇者候補
   - タイムテーブル案

4. **ターゲットペルソナの定義**
   - 具体的な参加者像
   - 抱えている課題
   - イベント参加後の変化

5. **収支計画の策定**
   - 予算規模
   - 収入源（チケット、スポンサー）
   - 支出項目

## 2. 開催背景と目的

### 背景
Web3やAIといった先端技術が急速に発展する一方で、学生が実際にその技術に触れ、専門家と対話できる機会は限られています。WeCreate3は、日本最大級の学生Web3コミュニティとして、この課題を解決するために本イベントを企画します。

### 目的
1. **学びの機会提供**: 業界第一線で活躍する専門家から最新トレンドを学ぶ
2. **コミュニティ形成**: 学生・社会人・企業が繋がる場を提供
3. **キャリア支援**: Web3・AI領域への就職・起業のきっかけを作る
4. **社会実装の促進**: 技術を実社会でどう活用するかの議論を深める

### WeCreate3のミッションとの関連性
WeCreate3は「Web3・メタバースの認識を広げ、新たな可能性を創出する」というミッションを掲げています。本イベントは、そのミッションを体現し、学生主体で最先端の技術トレンドを社会に発信する場となります。

## 4. ターゲットペルソナ

### ペルソナ1: 大学生（Web3初心者）
- **属性**: 大学2-3年生、文理問わず
- **課題**: Web3に興味はあるが何から始めれば良いか分からない
- **期待値**: 基礎知識を学び、同じ関心を持つ仲間と出会いたい
- **イベント後の変化**: WeCreate3コミュニティに参加し、勉強会に通い始める

### ペルソナ2: 大学院生・エンジニア（Web3経験者）
- **属性**: 大学院生、または社会人1-3年目のエンジニア
- **課題**: 技術は学んでいるが、ビジネス視点や社会実装例を知りたい
- **期待値**: 実務家の話を聞き、自分のキャリアに活かしたい
- **イベント後の変化**: 登壇企業でのインターンや就職を検討し始める

### ペルソナ3: 企業担当者（採用・新規事業）
- **属性**: Web3・AI関連企業の採用担当、新規事業担当
- **課題**: 優秀な学生人材とのタッチポイントが少ない
- **期待値**: 学生コミュニティとの接点を作り、採用候補を発掘したい
- **イベント後の変化**: WeCreate3と継続的な関係を構築し、次回スポンサーも検討

## Tips
- **ストーリーで語る**: 単なるスペックの羅列ではなく、「なぜこのイベントが必要か」をストーリーで語る
- **数値の根拠**: 参加人数や予算は過去実績や類似イベントを参考に現実的に設定
- **差別化ポイント**: 他のWeb3イベントとの違いを明確にする（学生主体、実践的、コミュニティ重視等）
- **ビジュアルも重要**: テキストだけでなく、イメージ図や過去イベントの写真を入れると伝わりやすい
- **更新を続ける**: 企画は固定ではなく、スポンサーや登壇者の状況に応じてアップデートしていく
