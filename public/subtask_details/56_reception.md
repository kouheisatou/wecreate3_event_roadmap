# 受付対応

## タスク概要
イベント当日、参加者をスムーズに受付し、名札・パンフレットを渡し、会場へ案内します。

## コピー可能なテンプレート

```markdown
# 受付対応マニュアル

**イベント名**: Web3 & AI 超会議 2026 Spring  
**開催日**: YYYY年MM月DD日（土）  
**受付時間**: 12:30-13:00（30分）

---

## 受付担当者

### 担当者（3名）
- **リーダー**: 〇〇
- **メンバー**: △△、□□

### 配置
- **受付1**: 〇〇、△△（一般チケット担当）
- **受付2**: □□（学生チケット担当）

## 受付フロー

### Step 1: 挨拶・確認
```
いらっしゃいませ！
Web3 & AI 超会議 2026 Spring へようこそ！

お名前をお伺いしてもよろしいでしょうか？
```

### Step 2: チェックイン

#### 方法1: Lumaアプリでチェックイン（推奨）
1. iPadでLumaアプリを開く
2. 参加者のQRコードをスキャン
   - 参加者がスマホでチケットQRコードを表示
   - iPadのカメラでスキャン
3. チェックイン完了

#### 方法2: 参加者リストで確認（バックアップ）
1. 紙の参加者リストで氏名を検索
2. チェックマークを付ける

### Step 3: 学生証確認（学生チケットのみ）
```
学生チケットをご購入の方は、学生証の確認をさせていただいております。
学生証を拝見してもよろしいでしょうか？

（学生証を確認）

ありがとうございます。
```

### Step 4: 名札・パンフレット配布
```
こちらが名札とパンフレットです。
名札にお名前をご記入ください。

（名札、パンフレット、ステッカーを渡す）
```

### Step 5: 案内
```
会場はこちらです。
お好きな席にお座りください。

本日はお楽しみください！
```

## トラブル対応

### ケース1: 参加者が見つからない
```
申し訳ございません。
もう一度お名前をお伺いしてもよろしいでしょうか？

（リストを再検索）

（見つからない場合）
恐れ入りますが、チケット購入時のメールを見せていただけますか？

（確認後、チェックイン）
```

### ケース2: 学生証を忘れた
```
学生証をお忘れですか。

恐れ入りますが、学生証の写真や、
学生メールアドレスでのメール画面など、
学生であることを証明できるものはございますか？

（証明できるものがあればチェックイン）
（証明できない場合）
大変申し訳ございませんが、学生チケットは学生証の確認が必須となっております。
一般チケットとの差額をお支払いいただけますでしょうか？
```

### ケース3: 予約していない（当日参加）
```
当日参加も可能ですが、満席の場合はご入場いただけない場合がございます。

現在の状況を確認いたしますので、少々お待ちください。

（統括に確認）

（席がある場合）
ご入場いただけます。
一般チケット ¥3,000 / 学生チケット ¥1,500 となります。

（満席の場合）
申し訳ございません。本日は満席となっております。
```

### ケース4: 遅刻（13:00以降に到着）
```
会場はこちらです。
すでに開演しておりますので、静かにご入場ください。

お座りになる際は、前方の空席にお座りください。
```

## 受付時間後の対応

### 13:00 開演後
- 受付は継続（遅刻者対応）
- 1名は受付に残り、2名は会場内の誘導へ

### 13:15 遅刻者対応終了
- 受付を閉める
- 全員が会場内の担当業務へ移動
```

## Tips
- **笑顔で挨拶**: 第一印象が大事
- **スムーズに**: 行列にならないよう、スムーズに対応
- **iPad充電**: バッテリー切れに注意
- **トラブルは冷静に**: トラブルが発生しても冷静に対応

## 所要時間目安
3時間（準備30分 + 受付30分 + 誘導2時間）
