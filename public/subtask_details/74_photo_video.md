# 写真・動画撮影

## タスク概要
イベント全体の様子を記録するため、写真・動画撮影を担当者を配置して実施します。事後のレポート作成、SNS発信、次回の告知素材として活用します。

## 目的
- イベントの様子を記録し、アーカイブ化する
- 事後レポートやSNS発信の素材を確保する
- 次回イベントの告知素材として活用する

## 実施手順
1. **撮影計画の策定**（1時間）
   - 撮影すべきシーン（受付、セッション、懇親会等）をリスト化
   - 撮影担当者を配置
   - 機材（カメラ、スマホ、三脚等）を確認

2. **撮影許可の確認**（30分）
   - 参加者への撮影許可取得（申込フォームで事前確認）
   - 登壇者への撮影・公開許可確認
   - スポンサーロゴの写り込み確認

3. **当日撮影**（イベント中）
   - 受付の様子
   - 会場全体の雰囲気
   - 登壇者のセッション
   - 参加者の表情
   - スポンサーブース
   - 懇親会の交流

4. **撮影データの整理**（イベント後・1時間）
   - ベストショットを選定
   - データをクラウドにアップロード
   - 共有URLを作成

## 成果物
- 写真データ（100-200枚）
- 動画データ（各セッション）
- 撮影データ共有URL

## 撮影時の注意事項

### 撮影許可
- [ ] 参加者への撮影許可を申込フォームで取得済み
- [ ] 登壨者への撮影・公開許可を確認済み
- [ ] 撮影NGの参加者がいる場合、受付で識別シールを配布

### 写り込み注意
- [ ] スポンサーロゴが適切に写り込んでいるか確認
- [ ] 参加者の顔が識別できる写真は事前許可を確認
- [ ] 機密情報やプライバシーに配慮

### 撮影テクニック
- [ ] 会場全体を撮る際は、参加者が多く見えるアングルで
- [ ] 登壇者は表情がわかるアップも撮影
- [ ] 懇親会は笑顔の瞬間を狙う
- [ ] 暗い会場では ISO感度を上げる

  - /[イベント名]/01_受付

  - /[イベント名]/03_懇親会

### 共有範囲

- スポンサー: ベストショット+スポンサー関連写真
- 参加者: ベストショット
- 一般公開: 選定した写真のみ

## Tips
- **担当者は複数人**: 1人だと撮り逃しが発生。2-3人で分担を
- **スマホでもOK**: 高画質なカメラが必須ではない。スマホでも十分
- **動画も撮る**: 写真だけでなく、短い動画（10-30秒）も雰囲気が伝わる
- **リアルタイム投稿**: 撮影した写真をその場でSNSに投稿すると盛り上がる
- **プライバシー配慮**: 顔が識別できる写真の公開は慎重に
- **スポンサーへの配慮**: スポンサーロゴが写り込むよう意識的に撮影

## 所要時間目安
3時間（計画1時間 + 当日撮影 + 整理1時間）
