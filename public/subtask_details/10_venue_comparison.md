# 会場候補リスト・比較表

## タスク概要
イベントに適した会場をリストアップし、条件を比較検討します。WeCreate3のようなWeb3学生団体では、都心アクセスの良さ、設備の充実度、予算のバランスを考慮します。

## 目的
- 複数の会場候補を比較検討する
- 最適な会場を選定する
- バックアッププランを用意する

## 実施手順
1. **会場要件の明確化**
   - 収容人数、日時、予算、立地、設備

2. **候補地のリストアップ**
   - Webサイト、過去イベント実績、紹介等から5〜10件ピックアップ

3. **空き状況の確認**
   - 電話またはフォームで問い合わせ

4. **比較表の作成**
   - 条件を一覧化して比較

5. **現地視察**
   - 最有力候補2〜3件を実際に見学

## 成果物
- 会場候補リスト・比較表

## 6. 現地視察チェックリスト

### 会場全体
- [ ] 収容人数は十分か（座席配置を確認）
- [ ] 受付スペースは確保できるか
- [ ] 搬入経路は問題ないか（エレベーターの大きさ等）
- [ ] 空調は十分か（満員時でも快適か）
- [ ] トイレの数は十分か

### 設備
- [ ] プロジェクターのスペック（解像度、明るさ）
- [ ] スクリーンのサイズ（全員から見やすいか）
- [ ] マイクの種類と本数（ワイヤレス、ピンマイク等）
- [ ] Wi-Fi速度（実測値を確認）
- [ ] 電源の位置と数（延長コードが必要か）
- [ ] 照明の調整（暗転、スポット等）

### 懇親会スペース
- [ ] 飲食物の持ち込みは可能か
- [ ] テーブル・椅子の配置は十分か
- [ ] BGMを流せるか（音響設備）
- [ ] ゴミ箱の設置場所

### その他
- [ ] 駐車場はあるか（機材運搬時）
- [ ] 近隣にコンビニ・飲食店はあるか
- [ ] 喫煙所はあるか
- [ ] 緊急時の避難経路

## Tips
- **複数候補を並行検討**: 第1候補が取れないこともあるので、必ず2〜3件並行で進める
- **学生団体割引を交渉**: 「学生主催」を前面に出すと割引してもらえることがある
- **スポンサー企業のオフィスも候補に**: 会場費がゼロになり、スポンサーのブランディングにもなる
- **現地視察は必須**: 写真と実際では印象が違うことが多いので、必ず現地を見る
- **Wi-Fi速度は実測**: 「Wi-Fiあり」と書いてあっても遅いことがあるので、現地でスピードテストを行う
- **キャンセルポリシー確認**: 万が一中止になった場合のリスクを把握しておく
