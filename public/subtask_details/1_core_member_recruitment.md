# コアメンバーの招集

## タスク概要

イベント運営の成功には、熱意と責任感を持ったコアメンバーの存在が不可欠です。このタスクでは、イベントの企画・運営を共に担うコアメンバー3〜5名を招集します。過去のイベント運営経験者や、イベントテーマに強い関心を持つ人材を優先的に声をかけます。信頼できる仲間を集めることで、その後のタスク遂行がスムーズになります。

## 目的

- イベント運営を担う信頼できるコアメンバーを確保する
- 多様なスキルセット（企画、営業、広報、制作など）を持つメンバーを集める
- 初期段階から運営に参加することで、メンバーの当事者意識を高める
- 長期的なコミットメントができるメンバーを選定する

## 実施手順

1. **必要な役割を洗い出す**（30分）
   - 統括、会場担当、広報担当、営業担当、制作担当など、必要な役割をリストアップ
   - 各役割に求められるスキルや経験を明確化

2. **候補者リストを作成する**（30分）
   - 過去のイベントで一緒に働いた信頼できる人
   - SNSやコミュニティで活発に活動している人
   - イベントテーマに関心が高い人
   - 推薦を受けた人

3. **個別に声をかける**（1時間）
   - メッセージや対面で、イベントの概要と参加のメリットを伝える
   - 期待する役割と作業量の目安を説明
   - コミットできる期間と時間を確認

4. **コアメンバーを確定する**（30分）
   - 3〜5名のコアメンバーを決定
   - 全員が揃ったタイミングでキックオフミーティングの日程調整

## 成果物

- コアメンバーリスト（名前、役割、連絡先、得意分野を記載）
- メンバーの合意確認記録

- **期待する作業量**: 週〇時間程度、主に[時期]に集中
- **メリット**: イベント企画・運営の実績、人脈形成、スキルアップ

## チェックリスト

- [ ] 必要な役割（統括、広報、営業、会場、制作など）を洗い出している
- [ ] 候補者リストを作成している（5〜10名程度）
- [ ] 各候補者に個別に声をかけ、イベント概要と期待役割を説明している
- [ ] コアメンバー3〜5名の合意を得ている
- [ ] メンバーの連絡先（Email、電話、LINE等）を確保している
- [ ] キックオフミーティングの日程調整を開始している

## Tips

- **信頼関係が最優先**: スキルよりも、最後までコミットしてくれる信頼できる人を選ぶ
- **役割は柔軟に**: 初期段階では大まかな役割分担でOK。途中で調整可能
- **多様性を意識**: 同じバックグラウンドの人だけでなく、多様な視点を持つメンバーを集める
- **早めに声をかける**: 優秀な人ほどスケジュールが埋まりやすいため、早期にアプローチ
- **メリットを明確に**: 無償ボランティアの場合、スキルアップや人脈形成などのメリットを伝える
- **小さく始める**: 最初は3名程度で始め、必要に応じて追加メンバーを募集する方法も有効

## 所要時間目安

2時間（役割定義30分 + 候補者リスト作成30分 + 声かけ1時間）
