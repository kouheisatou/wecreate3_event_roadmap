# スポンサー商談・提案

## タスク概要
スポンサー候補企業との商談を行い、資料を用いてメリットを説明し、契約の意向を確認します。

## コピー可能なテンプレート

```markdown
# スポンサー商談ガイド

**イベント名**: Web3 & AI 超会議 2026 Spring

---

## 商談前準備

### 企業リサーチ
- [ ] 企業の事業内容を理解
- [ ] 最近のニュース・プレスリリースを確認
- [ ] 採用ページで採用ニーズを確認
- [ ] SNSで発信内容を確認
- [ ] 競合他社の動向を確認

### 資料準備
- [ ] スポンサー募集Deck

- [ ] 契約書ドラフト
```

WeCreate3は、日本最大級の学生Web3コミュニティです。

2022年に設立し、現在500名以上のメンバーが在籍しています。

```
## 想定質問と回答例

### Q1: 過去の実績を教えてください
```
A: 前回の「Web3超会議 2024 Fall」では、
   参加者300名、満足度4.5/5.0という高い評価をいただきました。
   スポンサー企業様からも、「優秀な学生と直接話せて良かった」
   「費用対効果が高かった」というお声をいただいております。
```

### Q2: 参加者の属性は?
```
A: 参加者の60%が学生（情報系専攻、プログラミング経験あり）、
   25%が社会人（エンジニア、ビジネス職）、
   15%が企業担当者（採用、新規事業）です。
```

### Q3: どれくらいのブランド露出が見込めますか?
```
A: プレスリリース配信（PR TIMES、CoinDesk Japan）で累計5,000PV、
   SNS拡散で推定リーチ20,000人、
   イベントページで5,000PVを見込んでいます。
```

### Q4: 他にどんな企業がスポンサーですか?
```
A: 現在、[企業A]様、[企業B]様にご協賛いただいております。
   （守秘義務がある場合は「現在交渉中の企業様がございます」）
```

### Q5: 値引きは可能ですか?
```
A: 基本的にプラン通りの料金をお願いしておりますが、
   複数年契約や、現物協賛との組み合わせなど、
   柔軟に対応させていただきます。
   ぜひご相談ください。
```

### Q6: キャンセルポリシーは?
```
A: X週間前までは無料キャンセル可能です。
   それ以降は、規定のキャンセル料が発生します。
   詳細は契約書に記載しております。
```

## 商談Tips

### 準備が9割
- 企業リサーチを徹底
- 資料を完璧に準備
- 想定質問の回答を用意

### 相手目線で語る
- 「自分たちが嬉しい」ではなく「御社にとってのメリット」を語る
- データで語る（過去実績、参加者属性）

### 柔軟に対応
- プランのカスタマイズに柔軟対応
- 先方の予算に合わせて提案

### 熱意を伝える
- 学生の熱意は最大の武器
- 真摯に、丁寧に、熱意を持って

### フォローを怠らない
- 商談後のフォローが成約率を大きく左右
- しつこすぎず、適度な頻度でフォロー

---

**作成者**: 〇〇（スポンサー担当）
```

## Tips
- **事前準備が重要**: 企業リサーチ、資料準備を徹底
- **データで語る**: 過去実績を数値で示す
- **相手目線**: 企業にとってのメリットを明確に
- **フォローを忘れずに**: 商談後のフォローが成約率を左右

## 所要時間目安
8時間（商談準備1h + 商談30分 + フォロー30分）× 複数社
