# コンセプト詳細化

## タスク概要
イベントのコンセプトを深掘りし、「誰に、何を、何のために」提供するのかを明確に言語化します。これがイベント全体の軸となります。

## コピー可能なテンプレート

```markdown
# イベントコンセプト詳細化ワークシート

**イベント名**: Web3 & AI 超会議 2026 Spring

---

## 1. Why（なぜ開催するのか）

### 背景・課題認識
```
Web3・AI領域は急速に発展しているが、学生が最新動向を学び、
業界のプロフェッショナルと交流できる機会は限られている。

また、企業側も優秀な学生との接点を求めているが、
従来の採用イベントでは双方向の深い交流が難しい。
```

### WeCreate3としてのミッション
```
日本最大級の学生Web3コミュニティとして、
学生と業界をつなぎ、次世代のWeb3・AI人材を育成する。
```

### このイベントで実現したいこと
1. 学生がWeb3・AIの最新トレンドを学べる場を提供
2. 業界のトップランナーと直接交流できる機会を創出
3. 学生同士のコミュニティ形成を促進
4. 企業の採用・ブランディングを支援

---

## 2. Who（誰に向けたイベントか）

### メインターゲット（60%）
**Web3・AI領域に関心のある大学生・大学院生**

- 年齢: 18〜25歳
- 専攻: 情報系、工学系、経済系
- 特徴:
  - プログラミング経験あり
  - ブロックチェーン・AIの基礎知識あり
  - キャリアを模索中
  - 同じ関心を持つ仲間を求めている

### サブターゲット1（25%）
**Web3・AI業界で働くエンジニア・ビジネスパーソン**

- 年齢: 25〜35歳
- 職種: エンジニア、PM、ビジネス職
- 特徴:
  - 最新トレンドのキャッチアップ
  - 横のつながりを求めている
  - 転職・キャリアアップを検討中

### サブターゲット2（15%）
**採用・新規事業担当の企業関係者**

- 企業: Web3・AI・スタートアップ
- 目的:
  - 優秀な学生との接点
  - 採用母集団形成
  - 自社のブランディング

---

## 3. What（何を提供するのか）

### 提供価値1: 学び
- Keynote: 業界トップランナーの講演
- Panel Discussion: 多様な視点からの議論
- 最新トレンド、実践事例の共有

### 提供価値2: 交流
- Networking: 登壇者・参加者との直接対話
- 学生同士の横のつながり
- 企業担当者との名刺交換

### 提供価値3: 刺激・モチベーション
- Lightning Talk: 学生プロジェクトの発表
- 同じ志を持つ仲間との出会い
- 「自分もやってみたい」と思える体験

### 提供価値4: キャリア
- 企業ブース: 採用情報、インターン情報
- キャリア相談
- 業界理解の深化

---

## 4. How（どのように提供するのか）

### イベント形式
- **形式**: オフラインカンファレンス + Networking
- **規模**: 200〜300名
- **会場**: 都心（アクセス良好、六本木・渋谷・新宿）
- **時間**: 13:00〜18:00（5時間）

### セッション構成
1. **Keynote（30分）**: 業界トップランナーの講演
2. **Panel Discussion（60分）**: 4名のパネリストによる議論
3. **Lightning Talk（60分）**: 学生プロジェクト8組発表
4. **Networking（105分）**: 懇親会、フード・ドリンク提供

### 演出・体験設計
- スポンサーブース: 企業との直接対話
- 写真撮影: SNS映えする会場演出
- BGM: オープニング・休憩時間の雰囲気作り
- ノベルティ: ステッカー、パンフレット
```

```
## 6. 参加者の理想的な体験フロー

### Before（参加前）
1. SNSでイベント情報を見る
2. 「面白そう！」と興味を持つ
3. Lumaでチケット購入
4. リマインドメールで期待が高まる

### During（当日）
1. 受付で名札・パンフレットをもらう
2. Keynoteで業界トップの話に刺激を受ける
3. Panel Discussionで多様な視点に触れる
4. LTで同世代の挑戦に感銘を受ける
5. Networkingで登壇者・企業・参加者と交流
6. 名刺交換、連絡先交換
7. 「来て良かった！」と満足して帰る

### After（参加後）
1. アンケートに回答
2. SNSで感想をシェア
3. 登壇資料・アーカイブ動画を見る
4. WeCreate3 Discordに参加
5. 次回イベントにも参加したくなる

---

## 7. 他イベントとの差別化

### 従来のイベント
- 企業主催のピッチイベント → 企業目線、学生は傍観者
- アカデミックカンファレンス → 研究者向け、難しい
- 就活イベント → 企業説明会的、堅い

### 本イベントの差別化ポイント
1. **学生コミュニティ主催**: 学生目線、親しみやすい
2. **Web3×AI**: 両領域をカバー
3. **双方向交流**: Networking時間を長めに確保
4. **学生プロジェクト発表**: 同世代の挑戦が見られる
5. **大規模**: 200〜300名の大規模イベント

## 9. リスク・懸念事項

### リスク1: 集客が伸びない
- 対策: 早期告知、インフルエンサー協力、スポンサー企業からの告知

### リスク2: 登壇者が集まらない
- 対策: 早めの依頼、過去の繋がり活用、謝礼の用意

### リスク3: 当日トラブル（機材、Wi-Fi）
- 対策: リハーサル徹底、バックアップ機材準備
```

## Tips
- **Why→Who→What→Howの順で**: この順番で考えると筋が通る
- **具体的に**: 抽象的な言葉ではなく、具体的な体験を描く
- **チームで議論**: キックオフMTGで全員で議論し、合意形成
- **ブレない軸**: このコンセプトがイベント全体の意思決定の軸になる

5時間（ワークシート記入 + チーム議論 + ブラッシュアップ）
